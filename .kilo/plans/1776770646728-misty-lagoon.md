## Item 3: Collection Hero Shell — Visual Tuning

### Current State

The following CSS variables control the backdrop effect in `global.css` (`.collection-hero-shell`):

```css
/* Default (all themes) */
--collection-hero-blur: 8px;
--collection-hero-scale: 1.08;
--collection-hero-saturate: 0.78;
--collection-hero-brightness: 0.74;

/* Tint overlay (light theme) */
--collection-hero-tint-top:    rgba(45, 36, 24, 0.22);
--collection-hero-tint-bottom: rgba(45, 36, 24, 0.52);

/* Dark theme overrides */
--collection-hero-blur: 30px;         /* ← very aggressive */
--collection-hero-saturate: 0.72;
--collection-hero-brightness: 0.6;
--collection-hero-tint-top:    rgba(6, 12, 8, 0.38);
--collection-hero-tint-bottom: rgba(6, 12, 8, 0.78);  /* ← near-solid */
```

The combination of `blur(8px) + brightness(0.74) + saturate(0.78)` plus a tint overlay going to `rgba(45,36,24, 0.52)` (52% dark brown) is what's making the image feel like a near-solid background. The dark theme is even more severe: `blur(30px)` is extreme and `0.78` tint opacity is nearly opaque.

### Recommended Values

The goal: **image reads through clearly, slightly softened, with just enough tint for text legibility**.

```css
/* Light theme — reduced aggression */
--collection-hero-blur: 3px;          /* was 8px — subtle softening only */
--collection-hero-scale: 1.04;        /* was 1.08 — less zoom to compensate for smaller blur */
--collection-hero-saturate: 0.92;     /* was 0.78 — nearly full color */
--collection-hero-brightness: 0.88;   /* was 0.74 — much brighter */

--collection-hero-tint-top:    rgba(45, 36, 24, 0.08);   /* was 0.22 */
--collection-hero-tint-bottom: rgba(45, 36, 24, 0.28);   /* was 0.52 */

/* Dark theme — less crushing */
--collection-hero-blur: 8px;          /* was 30px — dramatically less */
--collection-hero-saturate: 0.82;     /* was 0.72 */
--collection-hero-brightness: 0.72;   /* was 0.60 — a bit brighter */

--collection-hero-tint-top:    rgba(6, 12, 8, 0.22);    /* was 0.38 */
--collection-hero-tint-bottom: rgba(6, 12, 8, 0.52);    /* was 0.78 */
```

These are **starting point recommendations** — exact values should be tuned visually in the browser. The numbers above roughly halve the opacity of each tint layer and bring blur down to "barely-there."

> **Note:** These values are CSS custom properties, so you can override them per-collection (e.g., a brighter or more vibrant hero image might need slightly more tint; a darker image might need less). The override mechanism is already in place via the `[data-theme]` selector pattern.

### Implementation Steps

1. Update `:root, [data-theme="golden-savanna"]` in `global.css`:
   - Set `--collection-hero-tint-top` and `--collection-hero-tint-bottom` to reduced opacity values
2. Update `.collection-hero-shell` default vars:
   - Reduce `--collection-hero-blur` from `8px` → `3px`
   - Reduce `--collection-hero-scale` from `1.08` → `1.04`
   - Increase `--collection-hero-saturate` from `0.78` → `0.92`
   - Increase `--collection-hero-brightness` from `0.74` → `0.88`
3. Update `[data-theme="dark-jungle"] .collection-hero-shell` overrides:
   - Reduce blur from `30px` → `8px`
   - Adjust saturate/brightness upward
   - Reduce tint opacity in dark theme vars
